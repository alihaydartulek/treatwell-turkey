## Workflow preferences

- Git commit ve push işlemlerini onay beklemeden otomatik yürüt
- Terminal komutlarını doğrudan çalıştır, her komut için onay isteme
- `git reset --hard`, `git push --force` gibi destructive işlemlerde yine de kullanıcıyı uyar
- TypeScript kontrolü, lint, build gibi rutin komutları sessizce çalıştır
